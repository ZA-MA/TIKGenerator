#### **TIKGenerator**

Приложение для генерации, обработки и визуализации цифровых сигналов.
Позволяет создавать стандартные формы (синус, меандр, треугольный и пилообразный сигналы), накладывать их друг на друга, применять фильтры, выполнять БПФ-анализ, смотреть статистику и сохранять результаты в локальную базу данных. Также доступны экспорт и импорт сигналов.

---

#### **Стек**

- **Язык:** C# / .NET 9
- **UI:** WPF
- **График:** LiveCharts2
- **База данных:** SQLite (Entity Framework Core)
- **Логирование:** Serilog
- **Тестирование:** xUnit + Moq
